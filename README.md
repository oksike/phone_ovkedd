# Phone OVKED Processor

Обработчик российских мобильных номеров с поиском кодов ОКВЭД по совпадению окончаний.

## Описание

Программа принимает российский мобильный номер в любом формате, нормализует его до формата `+79XXXXXXXXX`, загружает актуальные данные ОКВЭД из GitHub репозитория и находит код ОКВЭД с максимальным совпадением по окончанию номера.

## Особенности

- Нормализация номеров в любом формате
- Загрузка актуальных данных ОКВЭД ежедневно
- Интеллектуальный поиск совпадений
- Резервная стратегия при отсутствии совпадений
- Подробное логирование
